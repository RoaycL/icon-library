# Personal Icon Library

个人图标库，供 Surge、Loon、Egern 等工具使用。

- `icons/icons-all.json`：图标目录（1,609 项）
- `icons/cloudcone.png`、`icons/vmiss.png`、`icons/digitalocean.png`：个人新增图标

原始目录来自 [fmz200/wool_scripts](https://github.com/fmz200/wool_scripts) 的 `icons/icons-all.json`，根据其 GPL-3.0 许可证进行复制与修改；本仓库同样采用 GPL-3.0。
